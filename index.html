Instructions



Use [<-] & [->] keys to move the racket

> Avoid the ball passing through the racket's height. For each time it does, you lose a life.

> Press Spacebar to start the game. It will only start if your have entered a nickname

> If the ball touches the bottom of the game panel, it will stop.
Press Spacebar to place it above the racket and then press Spacebar again to continue

> When the game is over, your score will be displayed beneath


Gamemodes

  • Normal Mode: The game advances through all the levels sorted by difficulty. You can choose to start in whichever level you like

  • Timed Mode: Levels are ranom. Each level has a time designated. You must complete the level before the time runs out

  • Survival Mode: There's only 1 block. The game gets harder as time passes.
